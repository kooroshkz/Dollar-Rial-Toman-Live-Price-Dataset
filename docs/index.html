About This Dataset

This dataset provides comprehensive historical and live exchange rate data for USD to Iranian Rial (IRR) and Toman (IRT) from 2011 to the present day.

  • Source: TGJU.org (Tehran Gold and Jewelry Union)
  • Update Frequency: Daily via automated scraping
  • Data Points: Open, Low, High, Close prices
  • Currencies: Rial (IRR) and Toman (IRT)

Technical Details

  • Format: CSV with UTF-8 encoding
  • Date Format: DD/MM/YYYY
  • Persian Dates: Solar Hijri calendar
  • Toman Conversion: 1 Toman = 10 Rial
  • Automation: GitHub Actions daily updates
  • License: Open source
